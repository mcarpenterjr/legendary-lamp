Github Pages

Publish a copy of your repo to the web.

  • create the pages branch.
    • Click the branch button.
    • Choose create new branch.
    • Give the branch a name. In this case, pages.
    • Publish the new branch.
    • Go to the settings tab of your github account.
    • Choose the pages option.
    • Choose the new branch you just created and click save.
    • make sure "/root" is selected in the second dropdown of the source section.
